# RetailTalk Backend

FastAPI backend for RetailTalk — an NLP-powered e-commerce product search engine.

## Models included
- BERT multilingual embeddings
- Intent classifier
- Slot extractor (NER)
- CrossEncoder ranker
- ESCI classifier (Exact/Substitute/Complement/Irrelevant)
